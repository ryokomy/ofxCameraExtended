# ofxCameraExtended

This addon provides some useful cameras which inherit ofCamera.  
1. OrbitCamera  
2. MorphCamera  
3. FollowCamera  
4. FixedCamera  
5. EasyCamera  

Please refer to example for more information.

## dependency
- ofxEasing  
https://github.com/arturoc/ofxEasing
